# Ghumo India – Travel Companion App

Ghumo India is a mobile travel companion app built with MIT App Inventor that helps travelers plan and navigate trips across India. It brings essential travel information into one place, from discovering destinations to booking hotels, restaurants, and local transport.

## Overview

The app is designed to make traveling in India simpler for both domestic and international tourists. Ghumo India centralizes key travel workflows so users can:

- Find places to visit in different cities and states.
- Access booking options for hotels and restaurants.
- Discover and book local transport like Uber, Rapido, and other services.
- Use an integrated translator to bridge language gaps while traveling.

## Features

- **Discover destinations** – Curated lists of tourist spots and points of interest across India.
- **Accommodation booking links** – Shortcuts to hotel and stay booking platforms.
- **Food & restaurants** – Access to restaurant information and booking/ordering services.
- **Local transport access** – Quick links to ride‑hailing and vehicle booking apps (e.g., Uber, Rapido).
- **Translator utility** – Simple translation support to help users communicate and understand local language content.

## Tech Stack

- **MIT App Inventor** – Block‑based visual programming environment for building Android apps.
- **Android (APK)** – Built and exported as an Android application package for installation on devices.
- **External services** – Deep links to third‑party travel, booking, and transport platforms.

## Project Context

Ghumo India was developed as part of a group project , with the goal of creating a practical travel assistant for Indian routes and destinations. The project focused on:

- Simplifying trip planning for travelers.
- Combining multiple travel services into a single app experience.
- Showcasing how MIT App Inventor can be used to build real‑world Android apps without traditional code.
